# BasePHP

BasePHP is a small framework for minimalist.

Request
---------------

### input()

```php
$request->input($name, $default)
```

### get()

```php
$request->get($name, $default)
```

### post()

```php
$request->post($name, $default)
```

### cookie()

```php
$request->cookie($name, $default)
```

### method()

```php
$request->method($name)
```

### isMethod()

```php
$request->isMethod()
```

### isAjax()

```php
$request->isAjax()
```

### isConsole()

```php
$request->isConsole()
```
